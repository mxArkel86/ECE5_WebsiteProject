  PID stands for Proportional Integral Derivative. The purpose of the P is to provide immediate feedback for the robot, in converting the photoresistor values into an adequate turning circle for the robot to follow. Integral is useful for dampening sudden changes in the input values, allowing for gradual changes in direction. Meanwhile, derivative reacts stronly to quick changes in direction, meaning that sudden changes in direction can be accurately handled.
